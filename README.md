# Java To-Do List Application

A console-based To-Do List application built using Java and Object-Oriented Programming principles.

---

## Features

- Add Tasks
- View Tasks
- Mark Tasks Completed
- Delete Tasks
- File Persistence
- Input Validation

---

## Technologies Used

- Java
- OOP
- File Handling
- ArrayList
- BufferedReader & BufferedWriter

---

## Project Structure

```text
src/
 ├── app/
 ├── model/
 └── service/

data/
 └── tasks.txt
```

---

## How To Run

### Compile

```bash
javac app/Main.java
```

### Run

```bash
java app.Main
```

---

## Application Menu

```text
1. Add Task
2. View Tasks
3. Mark Completed
4. Delete Task
5. Exit
```
